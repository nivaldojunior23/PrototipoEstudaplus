Equações do 2º grau
⏱ 15 min de leitura ✏️ 8 exercícios 📊 Nível médio

Uma equação do 2º grau é uma equação polinomial em que a maior potência da incógnita é 2. Ela tem a forma geral:

ax² + bx + c = 0    (a ≠ 0)

Os valores a, b e c são chamados de coeficientes. O coeficiente a não pode ser zero — se fosse, a equação seria de 1º grau.

Exemplo: Na equação 2x² + 3x – 5 = 0, temos a = 2, b = 3 e c = –5.

Para resolver uma equação do 2º grau, usamos a fórmula de Bhaskara. Mas antes, precisamos calcular o discriminante, chamado de Δ (delta):

Δ = b² – 4ac

O valor do delta nos diz quantas raízes reais a equação tem:

• Se Δ > 0: dois valores reais e diferentes
• Se Δ = 0: dois valores iguais (raiz dupla)
• Se Δ < 0: não existem raízes reais

Se o delta for maior ou igual a zero, as raízes são calculadas por:

x = (–b ± √Δ) / 2a

Q.6
Qual é o valor do discriminante (Δ) da equação x² – 5x + 6 = 0?
AΔ = 1
BΔ = 25
CΔ = –1
DΔ = 49
✓ Resposta correta!
Para x² – 5x + 6 = 0, temos a=1, b=–5, c=6. Calculando: Δ = b² – 4ac = (–5)² – 4·1·6 = 25 – 24 = 1. A alternativa correta seria A.
